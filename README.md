<h3 align="center">
    ESLint config weclapp
</h3>

#### Installation
```shell
$ npm install @weclapp/eslint-config
```

#### Usage
The weclapp eslint-rules override parts of eslint's `eslint:recommend`.
```js
{
    "extends": "weclapp",
    "rules": {
        // Your custom, project-specific rules
    }
}
```
